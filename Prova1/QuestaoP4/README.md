        Dados quatro algoritmos (A, B, C e D) e suas respectivas fórmulas gerais, analise a complexidade de cada um deles e classifique-os em ordem crescente de complexidade. 
        A) O(n) = 4n² + 3n + 2
        B) O(n) = 2ⁿ + 34
        C) O(n) = 2√n+12
        D) O(n) = 3 + 5n log n

RESPOSTA:

D) O(n) = 3 + 5n log n    Mais complexo

A) O(n) = 4n² + 3n + 2      
C) O(n) = 2√n+12
